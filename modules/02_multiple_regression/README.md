# ISYE 4031 Module 02 JupyterLite: Multiple Regression

This folder is the student-facing JupyterLite source for Module 02. Build it from this directory with:

```powershell
jupyter lite build --lite-dir .
jupyter lite check --lite-dir .
jupyter lite archive --lite-dir .
```

The archive command produces `_output/software_jupyter_lite-jupyterlite.tgz`. Open `files/00_START_HERE.ipynb` first after launching the site. The `homework/` folder contains student homework templates.
