# ISYE 4031 Module 06: Box-Jenkins ARIMA

Open `files/00_START_HERE.ipynb` first. The notebooks are designed to run in JupyterLite with the Python (Pyodide) kernel.

This module covers nonseasonal and seasonal Box-Jenkins modeling with Python, including stationarity, differencing, ACF/PACF identification, ARIMA/SARIMA fitting, residual diagnostics, and forecasting.
